# GitHub Pages Deployment Guide

## Automatic Deployment
The repository is already configured for automatic deployment to GitHub Pages. When you push to the `main` branch, GitHub Actions will automatically:

1. Build the project using `npm run build`
2. Deploy the built files to GitHub Pages
3. Make the site available at: `https://nishchaydev.github.io/tristar-fitness-clean/`

## Manual Deployment (if needed)
If you need to manually trigger deployment:

1. Go to your GitHub repository: `https://github.com/nishchaydev/tristar-fitness-clean`
2. Click on "Actions" tab
3. Find the "Deploy to GitHub Pages" workflow
4. Click "Run workflow" button

## Repository Settings
Make sure GitHub Pages is enabled in your repository settings:

1. Go to repository Settings
2. Scroll down to "Pages" section
3. Under "Source", select "GitHub Actions"
4. The site will be available at: `https://nishchaydev.github.io/tristar-fitness-clean/`
